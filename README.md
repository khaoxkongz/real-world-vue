# real-world-vue

## Project setup

```bash
pnpm install
```

### Compiles and hot-reloads for development

```bash
pnpm run serve
```

### Compiles and minifies for production

```bash
pnpm run build
```

### Lints and fixes files

```bash
pnpm run lint
```

### Spin up little API server

```bash
pnpm dlx json-server --watch db.json
```
